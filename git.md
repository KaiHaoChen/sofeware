# Git常用命令

### Windows

#### 克隆仓库

- git clone "仓库地址"

#### 拉取服务器代码

- git pull

#### 提交代码到本地缓存区

- git add

#### 推送代码到本地库

- git commit -m "备注"

#### 获取状态

- git status
